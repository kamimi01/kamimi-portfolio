## 工夫したこと
* グローバルメニュを固定配置し、どのページにいてもすぐ別のコンテンツをみることができる
* レスポンシブ対応を行った
* コンテンツごとの切り口にsvgを用いて波線にした
* プロダクトのスクリーンショットの上にカーソルを当てると、別コンポーネントが現れる
* コンタクトのメールアイコンを押下すると、メールアドレスをクリップボードにコピーすることができる
* メールアドレスをコピーすると、ツールチップを表示する
