チラシde神経衰弱

URL N/A（未リリースのため）
ソースコード https://github.com/kamimi01/Summer-Memory-Game
説明 「お盆deハッカソン」（イベントレポート）で1日でチーム開発した、神経衰弱ゲームです。
通常の神経衰弱と異なり、トランプではなく夏祭りのチラシをカードに見立ててプレイします。
工夫したこと iOSアプリ開発が初めてのメンバーを考慮し、UIKitではなく初学者に馴染みやすいSwiftUIを採用して開発しました。
SwiftUIはUIKitほどの柔軟性がまだありませんが、複雑なUI/UXではなかったため、SwiftUIを採用しました。
夏祭りのチラシをカードに見立てる、罰ゲームで肝試し風の機能を追加するなど、「オンライン夏祭り」というコンセプトに沿った機能を開発しました。
使用した技術 Swift / SwiftUI / XCode / Figma
